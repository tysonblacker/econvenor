Source code

Code repository

eConvenor is open-source and the code is in our Github repo.

You can clone it and start hacking right now if you like.

Issue tracker

Take a look at eConvenor's Trac issue tracker to see what needs to be done next.

Feel free to take on anything you see there!

If your browser gives you a security warning for trac.econvenor.org, don't worry, it's just that the SSL certificate is for www.econvenor.org. Yuo can safely ignore the warning.
