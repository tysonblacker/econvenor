{% ifequal menu.tips 'dashboard' %} Dashboard eConvenor works best when you make your browser window as large as possible. Click the coloured bar at the top of each Dashboard box to go directly to that section. If you're wondering how anything works, look up the User Guide and FAQs under the Help menu. Found a bug? Please let us know so we can fix it. You can report bugs through the Feedback menu. {% endifequal %} {% ifequal menu.tips 'decisions' %} View decisions Decisions only appear here after the minutes of the meeting in which they were made are sent out. When you make a decision in a meeting, click the Add decision button and enter it in the box which appears. {% endifequal %} {% ifequal menu.tips 'create_agenda' %} Create agenda If you set a time limit for all agenda items, eConvenor will calculate the meeting's estimated end time. Your changes on this page are automatically saved every 60 seconds. If you run out of space in the Location box, put additional location information in the Instructions box. It's good practice to select a person to explain substantial agenda items. Do this using the To be explained by box. Navigate straight to an agenda item by clicking on it in the Navigation toolbar. Re-order agenda items by dragging them in the Navigation toolbar. Delete an agenda item by clicking the round "x" button on the right hand side of its title bar. {% endifequal %} {% ifequal menu.tips 'review_agenda' %} Review agenda This page shows you exactly what the agenda will look like when it is sent out. To make changes to what you see, click the Make changes button under the Actions toolbar. If you'd like to review the agenda with text-to-speech software, click the Open in PDF format button to get a PDF copy. {% endifequal %} {% ifequal menu.tips 'create_minutes' %} Create minutes To add the group's existing tasks to the end of the minutes, click the add a summary of existing tasks... box in Meeting details. Your changes on this page are automatically saved every 60 seconds. When the group makes a decision, click the Add decision button and enter the details in the box which appears. When a task is assigned to someone, click the Add task button and enter the details in the boxes which appear. Navigate straight to a minutes item by clicking on it in the Navigation toolbar. It's best practice to keep the Meeting notes as brief as possible, and focus on recording tasks and decisions. Items which come from the agenda can't be deleted in the minutes. Just enter "Not discussed" in the Meeting notes if necessary. {% endifequal %} {% ifequal menu.tips 'review_minutes' %} Review minutes This page shows you exactly what the minutes will look like when they are sent out. To make changes to what you see, click the Make changes button under the Actions toolbar. If you'd like to review the minutes with text-to-speech software, click the Open in PDF format button to get a PDF copy. {% endifequal %} {% ifequal menu.tips 'new_meeting' %} New meeting Once you've added a new meeting, you can make its agenda. If you run out of space in the Location box, put additional location information in the Instructions box. {% endifequal %} {% ifequal menu.tips 'current_meetings' %} Current meetings The red button shows the next task for each meeting. If you don't see the meeting you are looking for in this list, try looking under Archived meetings. Meetings are automatically archived once their minutes are sent out. {% endifequal %} {% ifequal menu.tips 'archived_meetings' %} Archived meetings Meetings are automatically archived once their minutes are sent out. To edit an archived meeting's agenda or minutes, first un-archive it via the Other actions button. {% endifequal %} {% ifequal menu.tips 'manage_participants' %} Manage participants Click on a participant to see their details and incomplete tasks. eConvenor automatically emails participants a reminder about nearly-due and overdue tasks. Use the Filter options to see only active participants, inactive participants or former participants. {% endifequal %} {% ifequal menu.tips 'new_participant' %} New participant Given name can be whatever you like: a real name, nickname, alias or nom de guerre. Make sure the email address is correct. This is how eConvenor sends agendas, minutes and reminders to participants. eConvenor automatically emails participants a reminder about nearly-due and overdue tasks. {% endifequal %} {% ifequal menu.tips 'edit_participant' %} Edit participant Given name can be whatever you like: a real name, nickname, alias or nom de guerre. Make sure the email address is correct. This is how eConvenor sends agendas, minutes and reminders to participants. eConvenor automatically emails participants a reminder about nearly-due and overdue tasks. {% endifequal %} {% ifequal menu.tips 'manage_tasks' %} Manage tasks Click on a task to see its details and edit it. Use the Filter options to see only overdue tasks, incomplete tasks or completed tasks. {% endifequal %} {% ifequal menu.tips 'edit_task' %} Edit task When a task is complete, change its Status to Completed. This stops reminders about it being sent out. You can assign a task to a different participant just by changing Person responsible. {% endifequal %}
